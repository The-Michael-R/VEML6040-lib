This code can be seen as example code for the library I wrote for the VEML6040 sensor from Vishay Semiconductors. This is a private project and not related to Vishay Semiconductors.

Using the veml6040 library to collect the luminance on all four channels (RGB and W) and print the values out every 10 seconds.

This code runs until stopped (e.g. ctrl-c).
